# 🌐 SocialSphere

A full-stack social media web application built using the MERN Stack. Users can create posts, follow other users, like and comment on posts, save posts, receive notifications, and manage their profiles. The project also includes a secure Admin Panel for managing users and posts.

---

## 📸 Features

### 👤 User Features

- User Registration & Login (JWT Authentication)
- Edit Profile
- Upload Profile Picture & Cover Photo
- Create Post (Multiple Image Upload)
- Edit & Delete Own Posts
- Like & Unlike Posts
- Save & Unsave Posts
- Comment System
- Delete Comments
- Search Users
- Follow / Unfollow Users
- Notifications
- Suggested Users
- Followers & Following List
- User Profile
- Saved Posts
- Responsive UI
- Dark Mode

---

### 🛠️ Admin Features

- Secure Admin Login
- Protected Admin Routes
- Dashboard Statistics
- View All Users
- Delete Users
- View All Posts
- Delete Posts
- Admin Logout

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/your-username/SocialSphere.git
```

---

### Backend Setup

```bash
cd server
npm install
npm run dev
```

---

### Frontend Setup

```bash
cd client
npm install
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env` file inside the **server** folder.

```env
PORT=5000

MONGO_URI=YourMongoDBConnectionString

JWT_SECRET=YourSecretKey
```
